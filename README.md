# AI Prompt Evaluation Portfolio

## Overview

This repository demonstrates my ability to evaluate AI-generated responses using structured quality assessment criteria.

As part of my journey into Artificial Intelligence and Language Quality, I created this portfolio to showcase practical skills used in AI training, prompt engineering, response evaluation, and quality assurance.

The examples included in this repository assess AI responses based on:

- Instruction Following
- Accuracy

These evaluation methods closely reflect workflows used in AI Data Annotation, AI Training, Language Quality Assessment, and Prompt Engineering.

## Evaluation Criteria

Every response is reviewed using a consistent rubric.

| Criterion | Description |
|-----------|-------------|
| Instruction Following | Did the AI follow every instruction? |
| Accuracy | Are the facts correct? |
| Clarity | Is the response easy to understand? |
| Completeness | Is important information missing? |
| Safety | Is the response appropriate? |
